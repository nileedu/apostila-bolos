# 🎂 Apostila Fatias de Bolos & Framework Landing Pages

Este repositório é uma base profissional replicável para criação de landing pages de vendas de alta conversão.

---

## 🚀 Como Iniciar um Novo Projeto

1. **Briefing**: Responda as perguntas em `documentos/briefing-novo-produto.md`.
2. **Escolha de Layout**: Visualize os modelos em `variacoes/` e escolha o que melhor se adapta ao nicho (Elegante, Direto ou Editorial).
3. **Desenvolvimento**: Siga as diretrizes de `documentos/estrutura-replicavel.md`.

---

## 📂 Estrutura do Repositório

- **`index.html`**: Versão publicada (Projeto Atual: Apostila Bolos).
- **`variacoes/`**: Modelos de layout base para replicação.
- **`ativos/imagens/`**: Banco de ativos padronizado.
- **`documentos/`**: Framework operacional e inteligência do projeto.
- **`_revisao/`**: Histórico de preservação e backups.

---

## 🛠️ Stack Técnica

- Vanilla HTML/CSS/JS (Zero dependências externas pesadas).
- Google Fonts & Icons.
- Mobile-First & Performance Optimized.

---

## 🛡️ Política de Preservação Permanente

Neste repositório, **nada é apagado sem registro**. Antes de qualquer alteração estrutural ou substituição de projeto:
1. Crie uma pasta em `_revisao/YYYY-MM-DD_HH-MM`.
2. Mova os arquivos originais para lá.
3. Só então aplique a nova versão na raiz.
