# 🎂 Apostila Fatias de Bolos — Matheus Araújo

Landing page de vendas otimizada para a conversão do infoproduto "Apostila Fatias de Bolos", focada em confeiteiras que atuam com delivery e encomendas.

---

## 📂 Estrutura do Repositório

A organização do projeto segue rigorosos padrões de sanidade e consistência:

- **`index.html`**: Arquivo principal da landing page (Single File).
- **`PROJETO.md`**: Documentação técnica e diretrizes de desenvolvimento.
- **`README.md`**: Guia de navegação e informações gerais.
- **`RELATORIO_DE_SANIDADE.md`**: Histórico de auditoria e saneamento do repositório.
- **`ativos/imagens/`**: Banco de imagens padronizado e otimizado.
- **`documentos/`**: Materiais de apoio, branding e diretrizes de copy.

---

## 🚀 Regras do Projeto

1. **Mobile-First**: A página deve ser impecável em dispositivos móveis.
2. **Performance**: Código limpo em Vanilla JS e CSS puro, sem dependências pesadas.
3. **Consistência**: Toda alteração de ativos deve refletir a nomenclatura oficial.
4. **Git**: Commits claros e descritivos seguindo o padrão estabelecido no `PROJETO.md`.

---

## 🛠️ Tecnologias Utilizadas

- HTML5 (Estrutura Semântica)
- CSS3 (Flexbox, Grid, Variáveis, Animações)
- JavaScript Vanilla (Interatividade e Carrossel)
- Google Fonts (Tipografia Premium)

---

## 🌐 Deploy

Este projeto está configurado para deploy automático na **Vercel**. 
Qualquer push na branch `main` atualiza a versão de produção instantaneamente.
