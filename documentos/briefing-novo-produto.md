# 📋 Briefing Obrigatório para Novos Produtos

Toda nova landing page criada neste repositório deve começar respondendo este briefing. A IA deve sugerir as respostas caso o usuário não as tenha.

---

## 🚀 Perguntas de Ativação

1. **Qual produto será trabalhado?**
2. **Qual é o nome oficial do produto?**
3. **Quem é o público-alvo (Ex: Mulheres, 25-45, iniciantes)?**
4. **Qual é a transformação ou promessa principal?**
5. **Quais são os módulos ou conteúdos do curso/apostila?**
6. **Quais são os bônus incluídos?**
7. **Quem é o especialista e qual sua autoridade?**
8. **Existem depoimentos reais ou resultados esperados?**
9. **Qual a faixa de preço e desconto aplicado?**
10. **Qual o estilo visual desejado?** (Elegante, Agressivo, Moderno, Minimalista, etc.)
11. **Há preferência de cores ou nicho?**
12. **Foco em qual pilar?** (Autoridade, Desejo, Urgência, Conversão Direta)

---

## 🎨 Sugestão de Direção (Tabela de Nichos)

| Nicho | Estilo Visual | Paleta Recomendada |
| :--- | :--- | :--- |
| **Confeitaria** | Gourmet / Apetite | Marrons, Dourado, Bege, Rosa Pastel |
| **Saúde/Bem-estar** | Limpo / Sereno | Verde Água, Branco, Azul Suave |
| **Finanças** | Autoridade / Seguro | Azul Marinho, Verde Escuro, Cinza Grafite |
| **Beleza/Estética** | Elegante / Feminino | Rosê Gold, Branco, Preto Luxo |
| **Marketing/Vendas** | Agressivo / Rápido | Vermelho, Preto, Laranja, Amarelo |

---
**Instrução para IA:** Após coletar estas informações, proponha pelo menos 3 variações de layout baseadas nos modelos da pasta `variacoes/`.
